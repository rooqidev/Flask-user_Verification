# Flask User Verification API

A small Flask API project with SQLite database for user registration and verification.  
Includes secure password hashing using PBKDF2 + salt.

## Features
- User registration
- User login with secure password verification
- SQLite database
- Version-controlled with Git

## Usage
1. Clone the repo
2. Install dependencies (`pip install -r requirements.txt`)
3. Run with Flask (`flask run`)
